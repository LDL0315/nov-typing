# 诺如病毒分型工具

一个基于本地 `reference/*.fa` 参考序列的静态前端分型页面，可提交 FASTA、多 FASTA 或原始核酸序列，分别对 VP1 与 RdRp 参考库做最佳重叠身份度匹配，并输出组合分型。

## 运行

```bash
python3 -m http.server 4173
```

然后打开：

```text
http://localhost:4173/
```

直接双击 `index.html` 时，浏览器可能会阻止页面读取 `reference` 目录，所以建议通过本地服务器访问。

## 分型逻辑

- 读取 `reference/nov-vp1-*.fa` 和 `reference/nov-rdrp-*.fa`。
- 自动解析每条参考序列标题中的基因型，如 `GII.4`、`GII.P17`。
- 对输入序列同时尝试正向与反向互补方向。
- 使用无缺口滑动重叠比对，按身份度、匹配数、重叠长度排序。
- 默认最小重叠长度为 `100 nt`，VP1 阈值为 `80%`，RdRp 阈值为 `85%`，可在页面中调整。
- VP1 与 RdRp 均通过时输出类似 `GII.P17_GII.17` 的双分型；仅一个片段通过时输出单片段结果。

## 文件

- 性能调整：页面设置中新增 k-mer 大小（默认 13）和候选数上限（默认 20），建议对全基因组使用 k=13、候选数 20。

- `index.html`：页面结构。
- `styles.css`：界面样式。
- `app.js`：FASTA 解析、参考库加载、分型和 CSV 导出。
- `reference/`：本地参考序列库。
